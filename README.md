# Shooting Range Game (Target Practice)

Shoot the moving targets to get points.  Bullseye is 10 points, each ring is 1 point less (outer ring is 1 point).  User's points (score) and shot count (shots) are displayed on the screen.

Try it out here! https://jryke.github.io/shooting-range-game/
